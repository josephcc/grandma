# Main


## Goal

This repo is a static website generator for a non-profit medical foundation. The name of the foundation is "財團法人臺北市林黃月英醫療基金會" or in English "Lin Huang Yueh-Ying Medical Foundation, Taipei, Taiwan"

The website has two stakeholders: 1) the users of the website; and 2) a non-technical website maintainer who edits the textual contents of the websites and posts update to it. The detail requirements are listed in `requirements.md`.

## Overview of planning and specification documents

This is the index file to the planning and specification documentation for this repository. Please refer to each of the following files when appropriate:

- `requirements.md` describes the high-level goals and functional requirements from the stand-point of the final user and the non-technical website maintainer.
- `tech_choices.md` describes the technology choices (i.e., libraries, languages, deployment platforms) for this project, the alternatives that were considered but not used, and the reasoning behind them.
- `architecture.md` describes the system architecture and deployment flow of the project.
- root/`maintainance.md` is a "README" file for the non-technical maintainer, which explains how to update the content of this project and triggers an deployment to the website.
